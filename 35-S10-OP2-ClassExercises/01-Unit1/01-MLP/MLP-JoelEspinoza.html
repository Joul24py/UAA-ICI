La Regla de Aprendizaje del Perceptrón¶

La idea general que hay detrs de la neurona de MCP (McCulloch Pitts) y del modelo del perceptrn umbralizado de Rosenblatt es utilizar un enfoque reduccionista para imitar cmo trabaja una simple neurona en el cerebro: si se excita o si no. As, la regla del perceptrn inicial de Rosenblatt es bastante sencilla y se puede resumir en los siguientes pasos:

  1. Iniciar los pesos a nmeros aleatorios

  2. Para cada muestra de entrenamiento $x(i)$:

    a. Calcular el valor de salida "$y$"

    b. Actualizar los pesos

La idea detrs de lo que queremos hacer es, con base en las cuatro variables que poseemos, determinar de qu tipo de flor se trata. Esto se har dndole un mayor o menor peso a cada variable usando la siguiente suma ponderada:

$$ (w_1 \cdot x_1) + (w_2 \cdot x_2) + (w_3 \cdot x_3) + (w_4 \cdot x_4) > \theta $$

Donde:

  • $x_1$ son valores pertenecientes a la columna s.largo
  • $x_2$ son valores pertenecientes a la columna s.ancho
  • $x_3$ son valores pertenecientes a la columna p.largo
  • $x_4$ son valores pertenecientes a la columna p.ancho
  • $w_1$ es el peso por el que se multiplicar $x_1$
  • $w_2$ es el peso por el que se multiplicar $x_2$
  • $w_3$ es el peso por el que se multiplicar $x_3$
  • $w_4$ es el peso por el que se multiplicar $x_4$
  • $\theta$ es el sesgo o bias que determinar la separacin lineal del problema

# Variables para graficar ms al rato y ver los resultados ms visualmente :)
erroresList = []
w1List = [w1]
w2List = [w2]
w3List = [w3]
w4List = [w4]
d1List = []
d2List = []
d3List = []
d4List = []

learnRate = 0.9
for i in range(10):
    errores = 0
    for j in range(len(df)):
        # Suma ponderada
        f = (w1 * df.loc[j, 's.largo']) + (w2 * df.loc[j, 's.ancho']) + (w3 * df.loc[j, 'p.largo']) + (w4 * df.loc[j, 'p.ancho'])
        
        # Comparamos la superacin del umbral
        if(f > theta):
            f = 1
        else:
            f = 0
        
        # Se contabilizan los errores
        if(f != int(df.loc[j, 'tipo'])):
            errores = errores + 1
            
            # Clculo de deltas
            d1 = learnRate * (int(df.loc[j, 'tipo']) - f) * df.loc[j, 's.largo']
            d2 = learnRate * (int(df.loc[j, 'tipo']) - f) * df.loc[j, 's.ancho']
            d3 = learnRate * (int(df.loc[j, 'tipo']) - f) * df.loc[j, 'p.largo']
            d4 = learnRate * (int(df.loc[j, 'tipo']) - f) * df.loc[j, 'p.ancho']

            # Actualizacin de pesos
            w1 = w1 + d1
            w2 = w2 + d2
            w3 = w3 + d3
            w4 = w4 + d4
            
    d1List.append(d1)
    d2List.append(d2)
    d3List.append(d3)
    d4List.append(d4)
    w1List.append(w1)
    w2List.append(w2)
    w3List.append(w3)
    w4List.append(w4)
    erroresList.append(errores)
print('Errores finales de la red: ' + str(errores))
